# 2026-06 NCIC 1:1 성취기준 검수 체크리스트

**목적**: 2022 개정 교육과정 성취기준 코드의 두 자리 번호(`-NN-NN`)를 NCIC 「2022 개정 평가기준」 PDF로 cross-check하고, 일치하면 `achievement_verified: true`로 승격.

**현재 상태**: 20/20 모두 `achievement_verified: false` (영역과 과목명은 본문 기반 추정 정확, 두 자리 번호만 검증 대기).

## 검수 절차

1. NCIC PDF에서 해당 과목 → 영역 → 두 자리 번호 확인
   - 통합과학·통합사회·공통국어·공통수학·한국사: NCIC 「2022 개정 공통과목 평가기준」
   - 정보: NCIC 「2022 개정 일반선택 평가기준」
2. 본 체크리스트의 `→ NCIC 코드` 칸에 실제 코드 기록
3. 일치 → 본 파일 체크박스 `[x]` + `tools/mark-verified.mjs --month 2026-06 --seq NN` 실행
4. 불일치 → 본 파일에 정확한 코드 기록 + JSON `meta.achievement_standard` 수정 후 재검증

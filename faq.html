구독 · 결제
구독은 어떻게 신청하나요? +

구독 신청 방법은 두 가지입니다.

  • 웹사이트 '구독 신청' 페이지에서 플랜과 레벨을 선택 후 신청
  • 카카오톡 채널 '테라 노바 영어'로 문의 후 직접 신청
첫 달 무료 혜택이 적용됩니다. 부담 없이 시작해보세요!
결제는 어떻게 이루어지나요? +

매월 구독일에 자동으로 결제됩니다. 신용카드, 체크카드, 카카오페이 등을 지원합니다. 결제 실패 시 카카오톡으로 안내 메시지가 발송됩니다.

3가지 플랜의 차이가 무엇인가요? +
  • PDF
    (14,900원)     — PDF 교재만 제공. 출력해서 쓰거나 태블릿으로 학습.
  • PDF+실물 (28,900원) — PDF + 매월 실물 책 배송 + 모의고사 2회 + 단어 암기장 + 시험지. 배송비 무료.
  • 프리미엄 (45,900원) — PDF+실물의 모든 것 + 입시뉴스 카톡방 + 해설강의(준비중).
플랜을 변경할 수 있나요? +

네, 가능합니다. 마이페이지에서 플랜 변경 신청을 하시면 다음 결제일부터 새로운 플랜이 적용됩니다. 당월 중에는 변경되지 않습니다.

배송
실물 교재는 언제 배송되나요? +

매월 1일~5일 사이에 일괄 발송됩니다. 택배사 사정에 따라 1~3일 내 수령하실 수 있습니다. 배송 시작 시 카카오톡으로 운송장 번호가 발송됩니다.

배송비가 있나요? +

PDF+실물 플랜과 프리미엄 플랜은 배송비 무료입니다. PDF 전용 플랜은 실물 배송이 없으므로 배송비가 없습니다.

제주도 및 도서산간 지역은 추가 배송비 3,000원이 발생할 수 있습니다.
배송 주소를 변경하고 싶습니다. +

마이페이지 → 배송 관리에서 주소를 변경하실 수 있습니다. 발송 전(매월 말일까지) 변경하셔야 당월에 적용됩니다. 이미 발송된 경우 변경이 불가능합니다.

교재를 받지 못했거나 파손된 경우 어떻게 하나요? +

카카오톡 채널로 문의 주시면 무료 재발송해드립니다. 파손된 경우 사진과 함께 보내주시면 빠르게 처리됩니다.

레벨 · 교재
어느 레벨을 선택해야 할지 모르겠습니다. +

레벨 테스트 페이지에서 3분 만에 진단받을 수 있습니다. 학년과 현재 영어 수준을 입력하면 최적의 레벨을 추천해드립니다.

→ 레벨 테스트 받기
레벨을 중간에 변경할 수 있나요? +

네, 가능합니다. 마이페이지에서 레벨 변경을 신청하시면 다음 달 발송분부터 새 레벨 교재로 배송됩니다. 레벨 변경에 따른 추가 비용은 없습니다.

교과 연계 학습이란 무엇인가요? +

Terra Nova의 모든 영어 지문은 실제 학교 교과서 단원과 직결됩니다. 예를 들어 중2 과학 '전기와 자기' 단원을 배울 때, 같은 주제의 영어 지문을 읽으며 영어와 과학을 동시에 학습합니다.

  • 수학·과학·사회·국어·기술 교과서 단원과 연계
  • 하루 2지문 × 5일 = 주 10지문, 월 40지문
  • 영어 점수 + 전과목 배경지식 동시 향상
PDF는 어디서 다운로드하나요? +

결제 완료 후 마이페이지 → 학습 자료에서 매월 PDF를 다운로드하실 수 있습니다. 발행일(매월 1일)에 자동으로 업로드됩니다. 카카오톡으로도 링크가 발송됩니다.

환불 · 해지
구독을 해지하고 싶습니다. +

마이페이지 → 구독 관리 → '구독 해지' 버튼을 클릭하시면 됩니다. 해지 신청 시 당월 만료까지 서비스를 계속 이용하실 수 있으며, 다음 달부터 결제되지 않습니다.

해지 후 재구독 시 첫 달 무료 혜택은 적용되지 않습니다.
환불 정책이 어떻게 되나요? +
  • 배송 전
    환불:     결제 후 실물 배송 전(매월 말일 이전)에는 전액 환불
  • 배송 후
    7일 이내:     교재 미사용 및 원상태 조건으로 부분 환불 (배송비 제외)
  • 배송 후
    7일 초과:     환불 불가
  • PDF: 다운로드 후 환불 불가 (미다운로드 시 전액 환불)
실수로 잘못된 레벨을 선택했습니다. +

배송 전(매월 말일까지) 카카오톡으로 문의 주시면 무료로 레벨을 변경해드립니다. 이미 배송된 경우 다음 달부터 변경 가능합니다.

환불·교환 정책

Terra Nova English 청약철회 · 환불 · 교환 · 반품 정책
시행일자: 2026년 1월 1일 · 본 정책은 「전자상거래 등에서의 소비자보호에 관한 법률」 및 「소비자기본법」을 따릅니다.

1. 청약철회 일반 원칙

회원은 상품·서비스 수령일(콘텐츠는 이용 가능 시점)로부터 7일 이내 청약철회를 요청할 수 있습니다. 단, 아래 항목별 세부 기준이 있는 경우 그에 따릅니다.

2. 항목별 환불 기준

구분환불 가능 여부 / 기준비고
디지털 콘텐츠 (LIGHT 플랜 / PDF·영상) 콘텐츠를 다운로드·열람·이용하지 않은 경우 7일 이내 전액 환불.
이미 다운로드/열람한 경우 「전자상거래법」 제17조 제2항 5호에 따라 환불 불가.		 이용 여부는 서버 로그로 확인
정기 구독 (월간) 결제일로부터 7일 이내 + 당월 콘텐츠 미수령: 100% 환불.
당월분 수령 후: 환불 불가, 다음 회차부터 자동 해지.		 마이페이지에서 즉시 해지 가능
정기 구독 (연간) 결제일로부터 7일 이내 + 어떠한 콘텐츠도 이용하지 않은 경우: 전액 환불.
이용 시작 후 중도 해지: 사용 월수 × 월 정가 + 위약금 10% 차감 후 잔액 환불.		 위약금은 「소비자분쟁해결기준」 기준
실물 상품 (STANDARD 책·시험지·단어장) 수령일로부터 7일 이내 미개봉 상태로 반품 시 전액 환불.
개봉·필기·훼손 시 환불 불가. 단순 변심 시 왕복 배송비 회원 부담.		 상품 하자 시 회사 부담
레벨 테스트 / 무료 샘플 무료 제공 항목으로 환불 대상이 아닙니다. —

3. 회사 귀책사유로 인한 환불

아래의 경우 회원 부담 없이 전액 환불되며 왕복 배송비도 회사가 부담합니다:

  • 상품의 결함·파손·인쇄 불량
  • 주문과 다른 상품이 배송된 경우
  • 배송 지연으로 인한 학습 진도 차질 (회사 귀책)
  • 서비스 장애로 콘텐츠 이용이 7일 이상 불가능한 경우

4. 환불 처리 절차 및 기간

  1. 환불 요청 접수 (이메일 또는 카카오채널)
  2. 요청 내용 확인 (영업일 기준 1~2일)
  3. 실물 상품의 경우 반품 수거 (영업일 기준 2~3일)
  4. 환불 처리 (영업일 기준 3~5일, 결제수단별 상이)
    • 신용카드: 카드 승인 취소 (카드사에 따라 1~2영업주 소요)
    • 계좌이체·간편결제: 등록 계좌로 입금

5. 교환 정책

  • 실물 상품의 단순 변심 교환은 미개봉 상태에서만 가능하며, 왕복 배송비는 회원 부담입니다.
  • 회사 귀책사유로 인한 교환은 회사가 모든 비용을 부담합니다.
  • 레벨 변경(예: STANDARD → ADVANCED)은 매월 1회 무료로 가능합니다. 마이페이지에서 직접 변경하세요.

6. 청약철회가 제한되는 경우

「전자상거래법」 제17조 제2항에 따라 다음의 경우 청약철회가 제한됩니다:

  • 회원의 사용·소비로 상품 가치가 현저히 감소한 경우 (실물 교재 필기·표시 등)
  • 시간 경과로 재판매가 곤란할 정도로 가치가 감소한 경우
  • 복제 가능한 콘텐츠를 다운로드·열람한 경우 (PDF·영상)

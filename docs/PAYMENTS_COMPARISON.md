# 결제 시스템 (PG) 비교 — Terra Nova

작성일: 2026-04-27
대상: 월간 구독 (LIGHT 11,900원 / STANDARD 24,900원 / PREMIUM 38,900원) + 연간 결제

## 우선 결정 기준

Terra Nova의 결제 요구사항:
1. **정기결제 (자동 빌링)** — 월간/연간 구독 자동 갱신 필수
2. **간편결제 지원** — 모바일 학부모 타깃이라 카카오페이/네이버페이 필수
3. **개발 부담 최소화** — 1인 운영
4. **수수료** — 학습지 단가가 낮아(11,900원) 수수료가 마진에 직결
5. **에스크로** — 통판 신고용. 이미 스마트스토어 가입했으면 별도 PG의 에스크로는 불필요

---

## 옵션 A: PortOne (포트원, 구 아임포트) ⭐ 추천

이미 `portone-client.js`가 코드베이스에 있어서 가장 빠른 길.

### 특징
- **결제 통합 SDK** — 여러 PG사를 하나의 코드로 연동
- 정기결제 (빌링키) 지원
- 카카오페이·네이버페이·토스페이먼츠·KG이니시스 등 통합
- **무료 시작** (월 거래액 따라 추가 비용 발생)
- 한국 스타트업/구독 서비스 표준

### 수수료 (PG사 통과분)
- 신용카드: 2.7~3.4% (PG에 따라)
- 간편결제: 2.0~2.9%
- PortOne 자체 수수료: **0원** (기본 플랜)

### 장점
- 이미 클라이언트 코드 구조 있음 → 재사용
- 한 번 연동으로 여러 결제수단 동시 지원
- 한국어 문서/지원 우수
- **테스트 환경 무료**, 가맹점 등록 전 통합 테스트 가능

### 단점
- 실제 결제 받으려면 결국 하위 PG사(KG이니시스/토스 등) 가맹점 신청 필요 → 심사 1~2주
- 1인 운영자에게 가맹점 심사가 부담

### Terra Nova 적용 권장 구성
```
PortOne (통합 SDK)
  └─ 토스페이먼츠 (메인 PG, 카드/계좌이체) — 심사 빠름·UX 우수
  └─ 카카오페이 (간편결제, 학부모 타깃)
  └─ 네이버페이 (간편결제, 검색 유입 사용자)
```

## 옵션 B: 토스페이먼츠 직접 연동 (PG 단독)

### 특징
- 토스가 직접 운영하는 PG. PortOne 없이 직접 연동.
- 정기결제 (빌링키) 지원
- 한국 스타트업 인기 (개발 문서 매우 친절)

### 수수료
- 신용카드: 2.5~2.9% (협상 가능)
- 간편결제(카카오페이): 2.0%
- 정산주기: T+3 (영업일 기준 3일 후)

### 장점
- 단일 공급사 → 운영 단순
- 개발 문서 한국에서 가장 우수
- 가맹점 심사 빠른 편 (3~7영업일)

### 단점
- 네이버페이는 별도 가맹점 등록 필요 (네이버페이는 토스페이먼츠로 처리 안 됨)
- 다양한 카드사 옵션은 PortOne보다 적음

---

## 옵션 C: 네이버페이 + 카카오페이 (간편결제 직접)

### 특징
- 각 간편결제 사업자에 직접 가맹 등록
- PG 거치지 않고 직접 결제 받음

### 수수료
- 네이버페이: 3.0~3.4% (네이버 스마트스토어 외 외부몰 기준)
- 카카오페이: 1.99~2.5%

### 장점
- **이미 스마트스토어 가입 중** → 네이버페이 가맹점 자동 활성화 가능성 ↑
- 학부모 타깃에 가장 친숙한 결제수단 (간편)
- 정산주기 빠름 (네이버페이 T+1)

### 단점
- 신용카드 직접 결제는 별도 PG 필요 (토스/이니시스 등 추가)
- 정기결제(자동결제) 설정이 PG 대비 복잡
- 수수료 협상 여지 적음

---

## 옵션 D: 카카오페이 정기결제 + 카드 PG 분리

학부모 타깃 + 1인 운영 최적화 구성:

```
카카오페이 (메인 정기결제, 학부모 80% 사용 가정)
  + 토스페이먼츠 (카드결제, 일부 비카카오 사용자)
```

수수료가 살짝 비싸도 운영 단순함이 1인 사업자에 유리.

## 추천 결정 트리

```
Q1. 월 거래액 예상이 100만원 이하인가?
  YES → 옵션 A (PortOne) — 무료 시작, 거래액 늘면 그대로 확장
  NO  → 옵션 B 또는 D 직접 연동, 수수료 협상

Q2. 카카오페이 비중이 60% 이상 예상인가? (학부모 타깃이면 보통 그렇다)
  YES → 카카오페이 정기결제 우선 + 카드는 토스페이먼츠
  NO  → PortOne 통합 (모든 결제수단 균등)

Q3. 개발 시간 4시간 이내로 끝내고 싶은가?
  YES → PortOne (이미 portone-client.js 있음, 재사용)
  NO  → 직접 연동도 가능
```

## ⭐ Terra Nova 최종 추천: PortOne + 토스페이먼츠 + 카카오페이

### 이유
1. `portone-client.js` 이미 작성되어 있음 → 즉시 재사용
2. 토스페이먼츠 가맹점 심사 가장 빠름 (3~5영업일)
3. 카카오페이는 학부모 타깃에 필수
4. 거래액 늘어도 같은 코드로 다른 PG 추가 가능

### 가입 순서 (이번 주)

| 순서 | 작업 | 소요 |
|------|------|------|
| 1 | PortOne 회원가입 (portone.io) | 10분 |
| 2 | 토스페이먼츠 가맹점 신청 (PortOne 안에서) | 신청 30분 + 심사 3~5일 |
| 3 | 카카오페이 가맹점 신청 (PortOne 안에서) | 신청 30분 + 심사 5~7일 |
| 4 | 네이버페이 직접 가맹 (선택) | 3~5일 |
| 5 | 결제 테스트 (개발자 모드, 실결제 전) | 2~4시간 |
| 6 | 라이브 전환 + 실결제 테스트 (1,000원 자가 결제) | 10분 |

### 가맹점 신청 시 제출 서류
- 사업자등록증 사본
- 통신판매업 신고증 (수리 후)
- 대표자 신분증
- 사업자 명의 통장 사본
- 사이트 URL 및 운영 계획서 (간단)
- 이용약관 / 개인정보처리방침 / 환불정책 (✅ 이미 작성됨)

### 코드 통합 위치
- `portone-client.js` → 결제 호출 모듈 (이미 존재)
- `order.html` → 구매 양식, 결제 버튼
- `subscription_detail_complete.html` → 정기결제 상세
- Supabase Edge Function → 결제 webhook 처리 (영수증, 결제 검증)

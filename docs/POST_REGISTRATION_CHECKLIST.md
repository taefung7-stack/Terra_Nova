## 🚨 자주 놓치는 함정

1. **사업용 계좌 미등록 가산세** — 홈택스 등록 안 하면 부가세 매입 인정 안 됨
2. **개인정보처리방침 미게시** — 과태료 최대 1천만원, 사이트 푸터 링크 필수
3. **통판신고증 미게시** — 사이트에 신고번호 표시 안 하면 시정명령 + 과태료
4. **약관 변경 시 사전 공지 누락** — 7일 전(중대한 변경 30일 전) 공지 안 하면 효력 없음
5. **PG 가맹 심사 통과 못 하면 결제 불가** — 사이트 약관/개인정보/환불정책 미비 시 반려 사유
6. **부가세 신고 누락** — 가산세 + 카드 매출 자동 신고분과 불일치 시 세무조사 가능
